Overview

Purpose

Consolidate all of my various design work into one comprehensive portfolio.

Audience

Potential employers/clients.

Dynamic elements

There will be some sort of hover animation on the navigation to indicate which page the user is on. There will be an option on the portfolio page to choose what content the user wants to view (photography, ux/ui, or graphic design). I already have a dynamic gallery for my photography. For the graphic design and the ux/ui, each project will be displayed as a card that the user can click on to view the whole case study.
